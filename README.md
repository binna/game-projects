### 🔧 공통 실행 방법
1. 프로젝트 실행 파일(`.zip`)을 다운로드한다
   
2. 압축을 해제한 뒤 `.exe` 파일을 실행한다
   > 💡 **Windows 보안 알림**이 뜨는 경우 :
   > `추가 정보` 클릭 → `실행` 버튼 선택
   > <br/>
   > ![image](https://github.com/user-attachments/assets/2e5e8964-c228-41b1-8325-684387966a38)
   

* 게임 종료는 `Alt + F4` 키 또는 `ESC` 키를 누른다
